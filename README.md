# README #

Features
---
* Minifies JavaScript

* Minifies CSS

* Minifies HTML

* Changes HTML to reference the minified files

* Runs on Windows using .NET, and Linux using Mono.

Usage
---
	StaticSiteSquisher.exe "C:\Path\To\Site\Files"
